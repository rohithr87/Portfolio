Real-Time Pothole Detection

Object detection system using Faster R-CNN with ResNet50-FPN backbone in PyTorch for automated road damage identification. Trained on 665 annotated images with 1,319 pothole instances, achieving 62.9% AP@50 and 76.5% Recall. Deployed as a Docker-based web app on Hugging Face Spaces.

  • Multi-scale feature extraction via Feature Pyramid Network (FPN) with transfer learning from ImageNet.
  • Data augmentation pipeline using Albumentations and OpenCV for improved model robustness.
  • Interactive Gradio interface with adjustable confidence threshold for real-time inference.
  • Containerized deployment using Docker on Hugging Face Spaces.
